第二节　40个发明原理

IP 1、分割原理

英文名称：

Segmentation、Segment、Fragmentation

原理描述：

分割原理是指这样一种过程：以虚拟的方式或真实的方式将一个系统分成多个部分，以便分解（/分开、分隔、抽取）或合并（/结合、集成、联合）一种有益的或有害的系统属性。在多数情况下，会对分隔后得到的多个部分进行重组（或集成），以便实现某些新的功能，并（或）消除有害作用。随着分割程度的提高，技术系统逐步向微观级别发展。

指导原则：

(1)、将一个对象[1]分成多个相互独立的部分；

例如：

l  将轮船的内部空间分成多个彼此独立的船舱；

l  在学生宿舍楼中，将同一层分成多个功能相同的小房间；

l  将一个物理硬盘划分为多个相对独立的逻辑硬盘；

l  将一个大型项目分为多个小项目；

l  内燃机的多个汽缸；

l  将学生分成不同的年级，不同的班级；

l  将水流喷为水雾；将物体变为粉末；将粉末变为气体。

(2)、将对象分成容易组装（或组合）和拆卸的部分；

例如：

l  组合家具；

l  暖气上的多个暖气片；

l  将巨型载重汽车分解成卡车及拖车；

l  在计算机程序设计中，将一个功能复杂的函数分为多个功能单一的函数；

l  将一根根铁轨连接起来，形成铁路；

l  建筑上常用的预制件和活动房屋；

l  在公司的组织结构上，可以使用模块化[2]的方法来实现公司的柔性化；

l  出版公司为了满足不同教师对教材的不同要求，推出了模块化教材，教师可以根据自己的需要从现有的章节、论文、案例中选取感兴趣的部分来组成自己的教科书。

(3)、增加对象的分割程度。

例如：

l  窗帘的演变：一整块布做的窗帘→左右两块布做的窗帘→百叶窗；

l  滚筒传送带的演变；

IP 2、抽取原理

英文名称：

Extraction、Extracting、Retrieving、Removing、Removal、Separate、Take out、Taking away

原理描述：

抽取就是从整体中分离出有用的（或有害的）部分（或属性）。抽取可以以虚拟方式或实体方式来进行。

指导原则：

(1)、从对象中抽取出产生负面影响的部分或属性；

例如：

l  最初的空调是一体机，工作时，压缩机会产生噪音。随着技术的发展，空调被分为室内机和室外机两部分。将压缩机放在室外机中，减少噪音对人的影响；

l  在巡航时，战斗机的两个副油箱挂在飞机下方，飞行中会优先使用副油箱中的燃油；在进入战斗前，抛弃副油箱，以减轻飞机的重量，增加飞机的机动性能；

l  从口腔（整个系统）中拔掉（抽取）一颗坏的牙齿（有害部分），以改善整个口腔的健康状况。

（2）、从对象中抽出有用的（/主要的、重要的、必要的）部分或属性。

例如：

l  用狗叫声作为报警器的报警声，而不用养一条真正的狗（将狗叫声从“狗”中抽取出来，作为有用的部分来单独使用）；

l  将稻草人作为“人”的代表放在稻田中（将人的外形从整个“人”中抽取出来）；

l  化学试验中的蒸馏、萃取和置换都是从混合物中抽取出有用物质的过程。

IP 3、局部质量原理

英文名称：

Local quality

原理描述：

在一个对象中，特殊的（特定的）部分应该具有相应的功能或条件，能够最好地适应其所处的环境，或更好的满足特定的要求。

指导原则：

(1)、将对象、环境或外部作用的均匀结构变为不均匀的；

例如：

l  在胚胎形成的过程中，一个受精卵经过不断的细胞分裂后变为由多个细胞组成的细胞团，此时每个细胞都是相同的。但是随着细胞分化的开始，不同的细胞会逐渐形成不同的组织和器官，这些不同的组织和器官会实现不同的功能，并最终形成一个完整的动物个体；

图3.2　从受精卵到胚胎的演化过程

l  在矿井中为了减少粉尘，常常利用喷水装置向采掘机和运煤机喷出圆锥状的水雾。水雾中的水滴越小，消除粉尘的效果就越好，但是另一方面，如果水滴太小的话，就很难迅速沉降下来（含有粉尘的小液滴会被工人吸入到肺里），妨碍正常的作业。解决方案就是用一层圆锥状的，颗粒较大的水雾包围在雾化较好的雾锥外围。这样以来，内层的小液滴负责吸附粉尘，外层的大液滴负责吸附内层的小液滴，而大液滴可以迅速的沉降下来，从而达到既消除粉尘，又能迅速沉降的目的；

图3.3　锥状水雾的结构

l  对金属表面进行渗碳处理，可以增加材料表面的硬度（而金属内部的特性并没有改变），从而提高其耐磨性能。

 (2)、让对象的不同部分具有不同的功能或特性；

例如：

l  带橡皮的铅笔（橡皮的功能是擦除痕迹，铅笔的功能是产生痕迹）；

l  图钉一头尖（便于刺入物体内），一头圆（便于人手施加压力）；

l  羊角锤（一头用来钉钉子，另一头用来起钉子）；

l  瑞士军刀（包含多种工具，如：螺丝起子，尖刀，剪子等，其功能各不相同）。

(3)、让对象的不同部分处于完成各自功能的最佳状态。

例如：

l  键盘上的各个健的位置和大小各不相同，使用频率较高的键位于最方便操作的位置，最常使用的键在体积上往往比其他键大（如：空格键和回车键）；

l  在食盒中设置间隔，在不同的间隔内放置不同的食物，避免相互影响；

l  将一个大房间用隔断分成多个具有不同功能的小房间（厨房、卫生间、卧室、客厅、储藏室等）；

IP 4、增加不对称性原理

英文名称：

Asymmetry

原理描述：

增加不对称性原理涉及到从“各向同性”向“各向异性”的转换，或是与之相反的过程。各向同性是指，无论在对象的哪个部位，沿哪个方向进行测量，都是对称的。各向异性就是不对称，是指在对象的不同部位或/和沿不同的方向进行测量，测量结果是不同的。通过将对称的（均匀的）的形式（/形状、形态、外形）或结构变为不规则的（/无规律的、不合常规的、不整齐的、不一致的、参差不齐的），可以增加不对称性。

指导原则：

(1)、将对象（的形状或组织形式）由对称的变为不对称的；

例如：

l  计算机的内存条、声卡、显卡、网卡的插槽都是采用不对称结构，这种不对称结构可以保证这些设备的正确插接；

l  坦克装甲的厚度，在不同部位是不同的。这种不对称结构既可以保证重点部位的高抗打击能力，又可以有效的减轻坦克的重量；

l  飞机机翼的上面和下面的弧线是不同的，这种不对称结构能够在气流的作用下产生上升的力；

l  从天平到杆秤的发展；

   

图3.4　天平和秤

l  利用防误法[3]进行设计构件，可以确保装配过程的正确性。通过在构件上设计凹槽、不对称孔和挡块，使安装方位在几何上“键合”。

(2)、如果对象已经是不对称的了，就增加其不对称程度。

例如：

l  为提高焊接强度，将焊点由原来的椭圆形改为不规则形状；

l  杠杆原理。当支点在中间的时候，两边长度相等，是对称的；当支点两边的长度不相等的时候，会出现杠杆作用；进一步增加这种不对称性，将会使杠杆作用更加明显，可以用很小的力撬起很重的物体；

l  锁及钥匙运用增加不对称性原理，来保证键合结构的惟一性。

IP 5、组合（/合并）原理

英文名称：

Consolidation、Combining、Joining、Merge、Merging

原理描述：

1.         组合（或合并）既可以是空间上的，也可以是时间上的。其目的是将两个或多个相邻的对象（操作或部分）进行组合或合并。

2.         在多种功能、特性或部分之间建立联系，以便产生一种新的、想要的或惟一的结果。通过对已有功能的组合，可以生成新的功能。

指导原则：

(1)、在空间上，将相似的（/相同的、相关的、同类的、接近的、时间上连续的）的对象加以组合（/合并）；

例如：

l  利用网络将多台计算机联结起来；

l  将多种机床的功能“集成”到一起，形成加工中心；

l  最早的机枪是由多个自动步枪组合而成的；

l  将多根不同颜色的油笔集成为一根多色油笔；

l  将多个单一插座集成到一起，组成插线板；

l  将电视机、录像机、收音机和录音机的功能集成到一起，形成家庭影院；

l  将多种单一的健身器材组合起来，组成多功能健身器。

(2)、在时间上，将相似的（／相关的、同类的、接近的、相同的、时间上连续的）的操作或功能加以组合（/合并）（最好是实现并行工作，以提高工作效率）；

例如：

l  将多种机床按照特定产品的工艺规划排列起来，形成一条流水线；

l  将多种单一功能的农业机械按照一定的顺序集成到一起，形成联合收割机；

l  冷热水混合龙头可同时放出冷水和热水，根据需求调节为所需的温度；

l  割草机带了一个袋子，旋转刀刃割下草以后，草就被放入袋子中，形成了先收割后收集的过程。

（3）、将具有不同（或相反）功能的对象合并（或组合）在一起，实现新的功能[4]。

例如：

l  将刀子、螺丝刀、剪刀、钳子等工具进行组合，形成瑞士军刀；

l  将榴弹发射器、光学瞄准器和步枪组合到一起；

l  将多种功能集成到一个芯片中；

l  带有橡皮头的铅笔（将橡皮和铅笔这两个具有相反功能的对象组合在一起）；

l  羊角锤（一头能钉钉子，一头能起钉子；将具有相反功能的对象组合在一起）。

IP 6、多功能性（/多用性、广泛性）原理

英文名称：

Universality、Universal、Multi-functionality

原理描述：

多功能性或通用性就是将不同的功能或非相邻的操作合并。使一个对象（例如，对象X）具备多项功能（例如，同时具备功能A、功能B、功能C…），从而消除了这些功能（例如，功能B）在其它（相关）对象（例如，对象Y具有功能A、对象Z具有功能B）内存在的必要性（进而裁减对象Y、Z中承担该功能的子对象)，结果就是对象X可以实现多个对象（例如，对象Y、对象Z…）的功能。使对象具备多用性，可产生在其他情况下不存在的机会及协力优势。

指导原则：

(1)、使一个对象能够执行多种不同的功能，从而使其他只具有单一功能的对象成为多余的，进而可以将其他对象裁减掉。

例如：

l  瑞士军刀可以提供多种功能；

l  具备多种技能的操作人员（复合型人才）；

l  家庭娱乐中心（具有录音机、CD机、电视机、录象机等功能）；

l  可调扳手是一种具有多用性的扳手（一把扳手可适合多种螺母）；

l  食品安全法也具有多功能性。它不仅能够督促国内企业提高其产品质量，而且可以用来设置贸易壁垒，将别国的食品拒之门外。

IP 7、嵌套原理

英文名称：

Nesting、"Nested doll"、Nested structures、Matrioshka[5]

原理描述：

通过递归的将一个对象放入另一个对象的内部，或让一个对象通过另一个对象的空腔而实现嵌套。嵌套是指彼此吻合、彼此组合、内部配合的性质。

指导原则：

(1)、将一个对象嵌入另一个对象，然后将这两个对象再嵌入第三个对象，依此类推；

例如：

l  俄罗斯套娃；

l  由从大到小的多个碗组成的“套碗”；

l  老式收音机或电视机上的拉杆天线；

l  可伸缩的单筒望远镜；

l  吊车的吊臂。

(2)、使一个对象穿过或处于另一对象的空腔；

例如：

l  自动铅笔的空腔中可以放多根备用的铅笔芯；

l  嵌套量规、量具；

l  汽车安全带收缩机构；

l  飞机起落架（飞机起飞后，起落架被收到飞机的机体内部）；

l  盒尺。

IP 8、重量补偿原理

英文名称：

Counterweight、Anti-weight

原理描述：

通过用一个相反的平衡力（浮力、弹力或类似的力）来阻遏（/抵消）一个不良的（/不希望有的）力。

指导原则：

(1)、将对象与另一个能提供上升力的对象组合，以补偿其重量；

例如：

l  飞艇利用浮力来补偿人和货物的重量；

l  用氢气球悬挂广告条幅（利用氢气球提供的上升力来补偿条幅的重量）；

l  在一捆原木中混杂一些泡沫材料，从而使原木捆更容易漂浮；

l  鱼可以利用其身体中的鱼鳔来实现上浮和下潜。

(2)、通过与环境的相互作用(利用空气动力、流体动力等)实现对象的重量补偿；

例如：

l  机翼在空气中运动的时候，机翼上方空气密度减小，下方空气密度增加，产生升力；

l  直升机的螺旋桨与空气发生相对运动时，可以提升上升力；

l  水翼船的水翼与水发生相对运动时，可以为船提供向上的力；

l  利用阿基米得定律，轮船可产生巨大的浮力。

(3)、利用环境中相反的力（或作用）来补偿系统的消极的（/负面的）属性。

例如：

l  利用船体周围的海水来冷却油轮中所装载的易挥发液体。

IP 9、预先反作用原理

英文名称：

Prior counteraction、Preliminary anti-action

原理描述：

预先了解可能出现的问题，并采取行动来消除出现的问题、降低问题的危害或防止问题的出现。

指导原则：

(1)、事先施加反作用力，以抵消工作状态下过大的和不期望的应力；

例如：

l  预应力钢筋混凝土（在灌注混凝土之前，拉伸钢筋，然后在拉伸状态把钢筋固定在模型里并注入水泥。当水泥硬化后，把钢筋两头松开，钢筋缩短并使水泥收缩，从而提高了钢筋混凝土的强度）；

l  预应力螺栓；

l  预应力构件。

(2)、对于某种既具有有害影响又具有有用影响的作用A，可以预先实施一种效果与A中的有害影响相反的作用B，利用B所具有的影响来降低或消除A所产生的有害影响。

例如：

l  内有波纹层的包装箱纸在制作时使其波纹层和表面层反向弯曲，当胶水干燥后，包装箱纸则达到平直状态。

(3)、对有害的作用或事件，预先采取相反的作用。

例如：

l  当建筑物着火的时候，如果某人要冲到建筑物里面去救人，通常先要用水将这个人的全身浇湿，这样就可以在短时间内防止其被火烧伤。

IP 10、预先作用原理

英文名称：

Prior action、Preliminary action

原理描述：

在真正需要某种作用之前，预先执行该作用的全部或一部分。

指导原则：

(1)、预先对某对象进行所需的改变，这种改变可以是完全的，也可以是部分的。

例如：

l  方便面；

l  建筑业中大量使用的预制件；

l  纸上预先印刷好的表格；

l  预先被打孔的邮票、包装袋；今天，好象已经没有人知道最早的邮票是以没有打孔的一整版的形式销售的。最早，用户不得不将邮票一张一张的剪下来，然后用胶水粘到信封上。

l  背面涂有不干胶的邮票；

l  计算机软件中，根据用户当前状态而弹出的软件菜单列表；

l  在大型机械设备的总装过程中，大量的使用了预先装配好的组件（例如，在汽车的总装线上，所需要安装的是一个已经装配好的发动机，而不需要在总装线上临时用零件组装出一个发动机）；

l  新生产出来的棉布被水洗以后，通常会“缩水”。如果用没有经过“缩水”的棉布做成衣服的话，用水洗过一次以后，就会变小，影响正常使用。因此，当棉布被纺织出来后，通常要进行预先缩水处理。这样以来，制造出来的衣物被水洗以后，就不会再缩水了。

(2)、将有用的物体预置，以便使其在必要时能立即在最方便的位置起作用。

例如：

l  制造业中的流水线；

l  商场内预先安置的灭火器；

l  战斗片中，进行阵地战的时候，战士们会预先将手榴弹的后盖打开（指导原则１），放在触手可及的地方（指导原则２）。

 


[1] 在多本英文和俄文TRIZ著作中，此处使用的是system或object，不仅可以表示具体的、有形的“物”（物体或产品），而且也可以表示抽象的、无形的”事”（组织方式、行为方式、流程）。但是在某些汉语著作中，使用的却是“物体”一词，只表示具体的“物”。本文在此使用的是“对象”一词，不仅表示“物”，也表示“事”。当我们将发明原理应用到“物”上的时候，就表示具体的、物理形式上的改变；当我们将发明原理应用到“事”上的时候，就表示抽象的、概念形式上的改变。

[2] 模块化是指一个系统的部件可以被分离并重新组织的程度。通过使产品模块化可以在特定输入组合的情况下增加可能配置的种类。

[3] Pokayoke (防误法, Mistake-Proofing)：本方法认为操作人员无法完全避免失误，因此要追求零错误的产品，必须采用该方法，使操作人员的失误不至于造成残次品。
