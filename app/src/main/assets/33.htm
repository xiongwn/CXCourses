第四节　使用标准解系统的流程

对于初学者来说，在使用标准解的过程中，最困难的就是如何选择合适的标准解来解决问题。为了便于初学者更好的使用标准解，TRIZ研究者们设计了一个流程图。这个流程图可以帮助初学者从包含了76个标准解的标准解系统中选择正确的标准解。

图7.5.1　使用标准解系统的流程图

具体步骤如下：

1.         完整地描述需要解决的问题（，得到问题描述），在分析问题（该问题称为初始问题（initial problem））、找到根本原因的基础上，构造问题模型（该问题模型称为初始问题模型（initial problem model））。

2.         将问题模型转换为以物场形式表达的物场模型（该模型被称为问题的物场模型，简称为物场模型，下同）。

注释1：完整的物场模型应该包含一个产品（S1），一个工具（S2），以及一个产品与工具之间的相互作用（F）。

3.         判断初始问题是否属于检测问题？

l  如果是检测问题，到步骤3.1；

l  如果不是检测问题，到步骤4.1；

3.1.   判断初始问题是否能够转化为检测问题？

l  如果初始问题能够转化为检测问题，则使用标准解系统中子类4.1中的标准解；

l  如果初始问题不能转化为检测问题，到步骤4；

注释2：如果将初始问题直接转化为测量问题太复杂，可以先将初始问题转化为检验问题，然后再转化为测量问题。

4.         判断物场模型是否完整？

l  如果物场模型不完整，先执行步骤4.1，然后到步骤5；

l  如果物场模型是完整的，直接到步骤5；

4.1.   判断物场模型中是否存在有害作用？如果不存在有害作用，到步骤4.2；

4.1.1.        根据问题描述，判断是否允许向物场模型中引入物质和场？

l  如果问题描述中允许引入物质和场，则使用标准解1.1.1~1.1.6，或使用标准解系统中4.2子类中的标准解；

l  如果问题描述中不允许引入物质和场，则使用标准解系统中的5.1、5.2和5.5子类中的标准解；

4.2.   根据问题描述，判断是否允许向物场模型中引入物质和场？

l  如果问题描述中允许引入物质和场，则使用标准解1.1.7、1.1.8和1.2.3；

l  如果问题描述中不允许引入物质和场，则使用标准解系统中5.1、5.2、5.5子类中的标准解；

5.         判断物场模型中是否存在有害作用？

l  如果存在有害作用，到步骤5.1；

l  如果不存在有害作用，到步骤6；

5.1.   根据问题描述，判断是否允许向物场模型中引入物质和场？

l  如果问题描述中允许引入物质和场，则使用标准解1.2.1、1.2.2、1.2.4、1.2.5；

l  如果问题描述中不允许引入物质和场，则使用标准解系统中5.1、5.2、5.5子类中的标准解；

6.         根据问题描述，判断物场模型中是否存在铁磁性物质？

l  如果物场模型中存在铁磁性物质，到步骤7；

l  如果物场模型中不存在铁磁性物质，到步骤8；

注释3：仔细检查一下，在技术系统的各个子系统中，是否存在任何可以被引入到物场模型中的铁磁性物质。

7.         根据问题描述，判断是否允许向物场模型中引入磁场？

l  如果允许向物场模型中引入磁场，到步骤17；

l  如果不允许向物场模型中引入磁场，到步骤8；

8.         根据问题描述，判断是否允许形成复杂物场？

l  如果允许形成复杂物场，则使用标准解系统中2.1子类中的标准解；

l  如果不允许形成复杂物场，到步骤9；

注释4：在问题描述中，如果对技术系统的复杂化没有限制，一般就可以通过形成复杂的物场来解决问题。

9.         根据问题描述，判断是否允许对物场模型进行替换？（用可控性更好的场来代替系统中原有的可控性差的场）

l  如果允许对物场模型进行替换，则使用标准解2.2.1；

l  如果不允许对物场模型进行替换，到步骤10；

注意5：除了磁场和电场以外，可以替换任何场。

注释6：在替换时，如果被引入的场会引起某种“副作用”，则这种替换是不能允许的。

10.     根据问题描述，判断系统是否是动态的？

l  如果系统是动态的，到步骤11；

l  如果系统不是动态的，则使用标准解2.2.2 ~ 2.2.4；

注释7：要时刻谨记“动态特性原理”，尽量增强技术系统的动态性；

11.     根据问题描述，判断物场模型中各个元素在结构上是否协调（互相匹配）？

l  如果是，到步骤12；

l  如果不是，则使用标准解2.2.5、2.2.6、4.3.1或标准解系统中5.3、5.4子类中的标准解；

注释8：记住这条法则的两重性！在某种情况下，有意识地使元素不协调也是必要的。

12.     根据问题描述，判断物场模型中各个元素的动态特性是否协调？

l  如果是，到步骤13；

l  如果不是，则使用标准解2.3.1 ~ 2.3.3，或4.3.2和4.3.3；

13.     根据问题描述，判断物场模型中是否允许引入铁磁性物质和磁场来替换现有的元素。

l  如果是，则使用标准解2.4.1或4.4.1；

l  如果不是，到步骤14；

14.     根据问题描述，判断是否允许向物场模型的现有物质中引入铁磁性添加物？

l  如果允许向物场模型的现有物质中引入铁磁性添加物，则使用标准解2.4.5或4.4.3；

l  如果不允许向物场模型的现有物质中引入铁磁性添加物，到步骤15；

15.     根据问题描述，判断是否允许向环境中引入铁磁性添加物？

l  如果允许向环境中引入铁磁性添加物，则使用标准解2.4.6或4.4.4；

l  如果不允许向环境中引入铁磁性添加物，到步骤16；

16.     根据问题描述，判断物场模型中是否允许使用电场和/或电流。

l  如果允许使用电场和/或电流，则使用标准解2.4.11和2.4.12；

l  如果不允许使用电场和/或电流，到步骤20；

17.     判断（铁）磁性物场模型是不是动态的？

l  如果（铁）磁性物场模型是动态的，到步骤18；

l  如果（铁）磁性物场模型不是动态的，则使用标准解2.4.2、2.4.3、2.4.4、2.4.7、2.4.8和4.4.2；

注释8：在步骤7中，我们只引入了磁场；在步骤17步中，我们利用（铁）磁性物场，使铁磁性物质具有动态特性（标准解2.4.2 ~ 2.4.4）或使物场模型中的所有元素具有动态特性。

18.     判断（铁）磁性物场模型中的元素在结构上是否协调？

l  如果是，到步骤19；

l  如果不是，则使用标准解2.4.9；

19.     判断（铁）磁性物场模型中各个元素的动态特性是否协调？

l  如果是，到步骤20；

l  如果不是，则使用标准解2.4.10、4.4.5和标准解系统中5.3、5.4子类中的标准解；

20.     在问题的解决方案中，按照下列顺序使用第三类标准解中的标准解：3.2.1、3.1.1、3.1.2、3.1.3和3.1.5；

注释10：标准解3.1.4可以被应用在双系统和多系统发展的任何阶段。

 

当然，类似的使用流程还有多种，本小节中所给出的只是其中的一种。希望读者能够以这个流程为参考，更好的学习和使用标准解系统来解决实际问题。
